Vault Vibe logo pack

Files included
- vaultvibe-logo-mark-master.png: transparent master logo mark
- vaultvibe-logo-mark-1024.png / 512 / 256: standard web-ready sizes
- favicon-16x16.png and favicon-32x32.png: browser favicons
- apple-touch-icon.png: iPhone/iPad home screen icon
- android-chrome-192x192.png and android-chrome-512x512.png: PWA/app icons
- vaultvibe-logo-preview-light.png: preview on a light background
- vaultvibe-logo-preview-dark.png: preview on a dark background
- vaultvibe-header-mark.png: padded mark for hero/header layout use

Recommended use
1. Use vaultvibe-logo-mark-master.png or 1024.png in slides, proposals, and design files.
2. Use 512.png or 256.png in websites and app UI.
3. Use the favicon and app icon files as-is for website metadata.
4. Keep generous spacing around the mark. Minimum clear space: about 15 percent of the logo width.
5. Prefer dark charcoal, black, silver, or clean white backgrounds so the metallic treatment stays sharp.
6. Do not stretch, rotate, recolor, or add heavy shadows.
7. For very small sizes, use the 32px and 16px favicon files only.

Suggested brand direction
- Primary dark: #0F172A
- Secondary dark: #111827
- Silver: #C7CDD4
- Light silver: #EEF2F6
- Black: #0A0A0A

Website placement
- Navbar: use a 32px to 44px rendered height
- Login / splash / hero: use 120px to 220px rendered height
- App dashboard sidebar: use 28px to 36px rendered height
- Slide deck cover: use 120px to 180px rendered height
